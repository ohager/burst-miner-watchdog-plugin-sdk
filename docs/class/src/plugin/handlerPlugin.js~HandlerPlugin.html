    HandlerPlugin

    Handler Plugin base class.

    A Handler receives events from the watchdog backbone. Based on that events you may trigger any action you want, i.e. send message on won block, etc. The watchdog can serve multiple handlers.

    Inherit from this class and override onEvent() for writing your own watchdog handler.
